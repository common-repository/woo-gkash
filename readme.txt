== Description ==
Gkash Payment for WooCommerce.
Gkash Payment gateway Checkout allows you to securely sell your producsts and subscriptions online using In-Context Checkout to help you meet security requirements without causing your theme to suffer.
In-Context Checkout uses a modal window, hosted on Gkash.My's servers, that overlays the checkout form and provides a secure means for your customers to enter their account information.

Also, with Integrated Gkash.My Setup (Easy Setup), connecting to Gkash.My is as simple as clicking a button , no complicated API keys to cut and paste, you just need to sign up as a Merchant with us at  https://www.gkash.my


== Installation ==
1. On the side bar of your dashboard, click Plugins and select "Add New".
2. Search "Gkash Payment" on the search plugins to find this plugin to install.
3. After the result from search, click Install Now to install the plugin to your wordpress.
4. Go to your install plugin and active the installed plugin.

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Gkash Payment, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Gkash Payment" and click Search Plugins. Once you’ve found our eCommerce plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.


== Frequently Asked Questions ==

= Who can use this plugin? =

Any merchant who sign up at  https://www.gkash.my

= What is the purpose for this plugin? =

This plugin provide you a new payment method for your website , so customers have more choices to choose for their payment method.
